# Practical Vibe Coding Course

이 저장소는 **Practical Vibe Coding Course**의 실습 코드와 문서를 관리하는 프로젝트입니다.

## 주요 기능 및 구성
- 바이브 코딩 실습 예제
- 다양한 웹 애플리케이션 및 모듈 구현

## 시작하기

### 사전 준비 사항
- 강의 시작 전 [강의전 수강생 준비사항](%EA%B0%95%EC%9D%98%EC%A0%84%20%EC%88%98%EA%B0%95%EC%83%9D%20%EC%A4%80%EB%B9%84.md)을 먼저 완료해 주시기 바랍니다.

### 설치 및 실행 방법
<!-- 차후 패키지 구성에 따라 빌드 및 실행 명령어는 업데이트될 수 있습니다. -->
```bash
# 의존성 패키지 설치
npm install

# 로컬 개발 서버 실행
npm run dev
```

## 규칙 및 약속
- 모든 코드의 주석은 **한글**로 작성합니다.
- 보안 정보가 담긴 `.env` 파일은 절대 Git에 커밋하지 않습니다. 대신 `.env-sample` 또는 `.env-example` 파일을 활용합니다.
- AI 에이전트(Antigravity) 작동 규칙 및 적용 방법은 [프로젝트 규칙(GEMINI.md)](GEMINI.md) 문서를 참고해 주세요.
