# Practical Vibe Coding Course

이 저장소는 **Practical Vibe Coding Course**의 실습 코드와 문서를 관리하는 프로젝트입니다.

## 주요 기능 및 구성
- 바이브 코딩 실습 예제
- 다양한 웹 애플리케이션 및 모듈 구현

## 시작하기

### 사전 준비 사항
- 강의 시작 전 [강의전 수강생 준비사항](강의자료/%EA%B0%95%EC%9D%98%EC%A0%84%20%EC%88%98%EA%B0%95%EC%83%9D%20%EC%A4%80%EB%B9%84.md)을 먼저 완료해 주시기 바랍니다.



## 규칙 및 약속
- 모든 코드의 주석은 **한글**로 작성합니다.
- 보안 정보가 담긴 `.env` 파일은 절대 Git에 커밋하지 않습니다. 대신 `.env-sample` 또는 `.env-example` 파일을 활용합니다.
- 이 프로젝트는 AI 에이전트(Antigravity) 작동 규칙이 자동으로 적용되도록 구성되어 있습니다. 상세 규칙 내용 및 직접 설정/수정하는 방법은 [프로젝트 규칙(GEMINI.md)](GEMINI.md) 문서를 참고해 주세요.
