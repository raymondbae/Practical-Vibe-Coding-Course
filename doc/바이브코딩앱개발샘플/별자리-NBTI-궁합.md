너는 트렌디하고 유머러스한 웹앱 개발을 돕는 데이터 엔지니어이자 전문 카피라이터야.
내가 "성향(NBTI) + 별자리 궁합 사이트"를 개발하려고 하는데, 여기에 들어갈 원천 데이터를 JSON 형식으로 생성해 줘. 
코드는 내가 직접 짤 테니, 오직 "순수한 데이터 구조(JSON)"와 "결과 문구 내용"에만 집중해서 작성해 줘.

톤앤매너는 20대~30대가 공감할 수 있는 힙하고, 위트 있고, 약간의 주접과 유머가 섞인 감성적인 톤으로 짜줘.

다음 세 가지 데이터를 JSON 배열 또는 객체 형태로 출력해 줘.

1. [별자리 궁합 데이터]
- 12개 별자리(양자리, 황소자리, 쌍둥이자리, 게자리, 사자자리, 처녀자리, 천칭자리, 전갈자리, 사수자리, 염소자리, 물병자리, 물고기자리) 간의 매칭 점수와 한 줄 평 데이터.
- 다 만들면 너무 많으니, 대표적인 조합 10개만 예시로 만들고 나머지는 내가 채울 수 있게 스키마 구조를 명확히 해줘.
- 형식 예시:
{
  "zodiac_matches": [
    { "signA": "양자리", "signB": "쌍둥이자리", "score": 85, "desc": "불과 공기의 만남! 대화가 끊이지 않는 유쾌한 바이브." },
    ...
  ]
}

2. [NBTI 조합별 케미 문구 데이터]
- NBTI의 4가지 지표(E/I, N/S, F/T, P/J) 중 "같은 글자의 개수"에 따른 궁합 점수와 찰떡 가이드 문구.
- 일치 개수 0개, 1개, 2개, 3개, 4개 총 5가지 케이스에 대한 유머러스한 코멘트.
- 형식 예시:
{
  "nbti_scores": {
    "4": { "score_name": "영혼의 쌍둥이", "desc": "숨만 쉬어도 서로 무슨 생각 하는지 아는 도플갱어 바이브! 밤새 수다 떨어도 모자라요." },
    "3": { ... },
    ...
  }
}

3. [최종 점수대별 한 줄 요약 문구]
- (별자리 점수 + NBTI 점수)를 합산한 최종 점수(0~100점) 대별 종합 코멘트.
- 0~30점(파멸의 바이브), 31~60점(밀당의 바이브), 61~85점(설렘의 바이브), 86~100점(우주가 점지한 운명) 총 4개 구간으로 나누어 작성.
