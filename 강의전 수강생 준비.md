# Practical Vibe Coding Course - 수강생 사전 준비사항

본 강의는 인공지능 기반의 코드 생성 및 협업 도구를 활용하는 실습 중심의 과정입니다. 원활한 실습 진행을 위해 **강의 시작 전**까지 아래의 준비 사항을 반드시 완료해 주시기 바랍니다.

---

## 1. 필수 소프트웨어 설치

### 1) Node.js 및 npm 설치
- 실습 프로젝트 빌드 및 실행을 위해 **Node.js (LTS 버전, 18.x 이상 권장)**를 설치해 주세요.
- **[공식 다운로드 페이지](https://nodejs.org/ko)**에서 OS에 맞는 인스톨러를 받아 설치하실 수 있습니다.
- 설치 완료 후 터미널(혹은 명령 프롬프트)에서 아래 명령어를 실행하여 설치 상태를 확인합니다.
  ```bash
  node -v
  npm -v
  ```

### 2) Git 설치 및 계정 설정
- 소스 코드 버전 관리를 위해 Git 설치가 필요합니다.
- **[Git 공식 다운로드 페이지](https://git-scm.com/downloads)**에서 다운로드 및 설치해 주세요.
- 설치 완료 후 사용자 이름과 이메일 설정을 진행해 주시기 바랍니다.
  ```bash
  git config --global user.name "본인이름"
  git config --global user.email "이메일주소"
  ```

### 3) Visual Studio Code (VS Code) 설치
- 주 개발 에디터로 VS Code를 사용합니다.
- **[VS Code 공식 웹사이트](https://code.visualstudio.com/)**에서 다운로드하여 설치해 주세요.
- 추천 확장 플러그인:
  - **Korean Language Pack** (한국어 팩)
  - **GitLens** (Git 정보 확인)
  - **Prettier** (코드 포맷터)

---

## 2. GitHub 계정 준비
- 실습 코드 저장소 연동 및 협업을 위해 **[GitHub 계정](https://github.com/)**이 필요합니다.
- 계정이 없으신 분은 사전에 가입하여 로그인 상태를 유지해 주시기 바랍니다.
